Hello! Welcome to use PDFPARSER~

PDFPARSER是一款pdf内容识别软件，提取其中文本内容，标题和表格，将其转化为标记的文本文件，并进而进行下一步的内容分析或者数据分析。PDFPARSER的关键特征是
  • 准确识别pdf中的表格元素，识别其表格中的每一行每一列，并将其采取“,”进行分割;
  • 采取API的方式进行调用，支持编程的方式进行大批量的pdf文件处理;
我们相信未来大规模处理PDF文件会成为常态，PDFPARSER愿与你同行。

使用说明

1.使用curl命令
  • 用户注册
    $curl -i -X POST -H "Content-Type: application/json" -d '{"username":"username","password":"password"}' http://pdf.simplified.org.cn:3000/api/users
  • 调用API解析文件
    $curl -u username:password -i -X POST -F "file=@filename.pdf" http://pdf.simplified.org.cn:3000/api/vbeta
